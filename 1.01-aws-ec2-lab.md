# 🧪 LAB: Create Linux and Windows EC2 Instances

---

# 🎯 Lab Objective

Create:

* `studentname-linux-vm`
* `studentname-windows-vm`

Using:

* Free tier eligible instance
* Proper security group
* Key pair
* Public access

---

# 🔹 Prerequisites

* AWS Account
* Login to AWS Console
* Select a Region (example: Mumbai)

---

# 🐧 PART 1 — Create Linux EC2 Instance

---

## Step 1: Go to EC2

1. Login to AWS Console
2. Search **EC2**
3. Click **Launch Instance**

---

## Step 2: Name the Instance

Name:

```
studentname-linux-vm
```

Example:

```
saravana-linux-vm
```

---

## Step 3: Choose AMI

Select:

* Amazon Linux 2023 (Free tier eligible)

---

## Step 4: Choose Instance Type

Select:

```
t2.micro
```

(Free tier eligible)

---

## Step 5: Create Key Pair

Click **Create new key pair**

Name:

```
studentname-key
```

Download `.pem` file.

⚠️ Save safely — cannot download again.

---

## Step 6: Configure Network Settings

Click **Edit**:

* Auto-assign Public IP → Enable
* Create new Security Group

Add Inbound Rules:

| Type | Port | Source   |
| ---- | ---- | -------- |
| SSH  | 22   | My IP    |
| HTTP | 80   | Anywhere |

---

## Step 7: Storage

Keep default:

```
8 GiB gp3
```

---

## Step 8: Launch Instance

Click **Launch Instance**

Wait until:

```
Instance state → Running
```

---

## Step 9: Connect to Linux VM

Option 1: EC2 Instance Connect (Browser SSH)

OR

Option 2: From your local system:

```bash
chmod 400 studentname-key.pem

ssh -i studentname-key.pem ec2-user@<Public-IP>
```

---

## Step 10: Test Web Server (Optional)

Install Apache:

```bash
sudo yum install httpd -y
sudo systemctl start httpd
```

Open browser:

```
http://<Public-IP>
```

You should see Apache page.

---

# 🪟 PART 2 — Create Windows EC2 Instance

---

## Step 1: Launch New Instance

Click **Launch Instance**

---

## Step 2: Name It

```
studentname-windows-vm
```

Example:

```
saravana-windows-vm
```

---

## Step 3: Choose AMI

Select:

* Microsoft Windows Server 2022 Base (or) Microsoft Windows Server 2025 Base

---

## Step 4: Instance Type

Select:

```
t2.micro
```

---

## Step 5: Key Pair

Use the same:

```
studentname-key
```

---

## Step 6: Network Settings

Inbound Rules:

| Type | Port | Source |
| ---- | ---- | ------ |
| RDP  | 3389 | My IP  |

---

## Step 7: Storage

Default:

```
30 GiB
```

---

## Step 8: Launch

Wait until:

```
Running
```

---

## Step 9: Connect to Windows

1. Select instance
2. Click **Connect**
3. Choose **RDP client**
4. Click **Get Password**
5. Upload your `.pem` key
6. Decrypt password
7. Open Remote Desktop
8. Connect using:

   * Public IP
   * Username: Administrator
   * Decrypted Password

---

# 🧹 Cleanup (VERY IMPORTANT)

To avoid charges:

1. Select both instances
2. Click **Terminate**
3. Delete unused Elastic IP (if created)

---

You should now understand:

* Key pair
* Security group
* Public IP
